
Temperature Converter
This project is a simple temperature converter that converts values from Fahrenheit to Celsius. The application is developed using Angular, allowing users to input a temperature in Fahrenheit, click a button to calculate, and display the result in Celsius.

Features
Data Input: Users can enter a temperature in Fahrenheit.
Calculation: By clicking the "Calculate" button, the temperature is converted to Celsius.
Result Display: The converted temperature is shown on the screen.
Technologies Used
Angular: For developing the application.
CSS: For styling the interface.
Instructions for Use
Clone the repository.
Install the necessary dependencies.
Run the application using the command ng serve.
Access http://localhost:4200 in your browser.
